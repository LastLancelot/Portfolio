# NestJS Task Management API

A backend REST API built with NestJS, TypeScript, TypeORM and JWT Authentication.

## Features

- JWT Authentication
- User Registration
- User Login
- Protected Routes
- Task Management
- PostgreSQL Integration
- Docker Support

## Tech Stack

- NestJS
- TypeScript
- PostgreSQL
- TypeORM
- JWT
- Docker

## Getting Started

### Environment Variables

Copy the example file:

```bash
cp .env.example .env
```

### Run with Docker

```bash
docker compose up --build
```

Application:

```text
http://localhost:3000
```

### Run Locally

```bash
npm install
npm run start:dev
```
