# Express REST API

A RESTful API built with Express and TypeScript.

## Features

- CRUD Operations
- Request Validation
- Authentication Middleware
- File Uploads
- Logging
- Error Handling

## Tech Stack

- Node.js
- Express
- TypeScript
- Multer
- Docker

## Running Locally

```bash
npm install
npm run dev
```

## Running with Docker

```bash
docker compose up --build
```

## API Modules

- Students
- Groups
- Authentication
- File Uploads
